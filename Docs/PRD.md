PRODUCT REQUIREMENT DOCUMENT (PRD)

Problem Statement
Users need a platform to upload, watch, and interact with videos easily, similar to YouTube, but with a clean, scalable architecture suitable for modern development.

Target Users
Students / Creators
Developers building portfolios
General video consumers

Goals
Build a full-stack scalable video platform
Provide smooth video consumption experience
Enable user interaction (likes, comments, subscriptions)
Maintain clean UI and modular backend

Core Features
Authentication (JWT + refresh tokens)
Video upload & management
Comments & likes system
Playlists
Subscriptions
Dashboard analytics

User Stories
As a user, I can register/login securely
As a creator, I can upload and manage videos
As a viewer, I can like, comment, and subscribe
As a user, I can create playlists
As a creator, I can view my channel stats

Success Metrics
Daily active users (DAU)
Video uploads per day
Average watch time
User engagement (likes/comments)
